üìù Notes

‚ñ∂
1. Symbols for each bitwise operators to be noted correctly: & (not &&- that's logical operator), XOR has ^, Bitwise Not has symbol ~ different than ! for not. 2.Bitwise left shift and right shift. Each number can be represented in binary. It's expressed as 1 or 0 × 2&sup{n}⊃ where n > 0 upto muchever necessary. So - 0 = 2&sup{0}⊃×0 = 0, 1 = 2&sup{0}⊃= 1, 2=1× 2&sup{1}⊃ + 0× 2&sup{0}⊃ = 10 And likewise, 3 = 11, 4 = 100, 7 = 111(2&sup{2}⊃+2&sup{1}⊃+2&sup{0}⊃), 8=1000 and so on. Now, 1 = 00000001 you can add as many zeros to left. So in bitwise left shift: All the binary digits shift to left by specified amount, empty spaces filled with zeros. This is equivalent of multiplying number by 2&sup{n}⊃. Example: Consider 5 (0001001). If we bit shift left by 1 digit, it becomes 0010010 = 5×2= 10.(0+2+0+0+8) Similiarity by 2 digits, it becomes 0100100 = 20. (0+0+4+0+0+16). Syntax: Result = operand << n, where n is number of shifting, it can be set empty for 1. So int a = 5; b = a << 2; printf("%d",b) Output: 20. (10100). Right shift is opposite, all binary digits shift to right, empty spaces filled with zeroes, equivalent of dividing number by 2&sup{n}⊃. Though this is different for say signed, unsigned bits, the general idea is same. Syntax same just use >>.

Input Format:

  • The first line contains two space separated integers, row & col representing the number of rows and columns of each matrix
  • The second line contains row * col number of space separated integers representing the elements of matrix 1
  • The last line contains row * col number of space separated integers representing the elements of matrix 2

  • Print row number of lines with col number of space separated elements representing the elements of sum matrix

üíª Answer Code

‚ñ∂ 

#include<stdio.h>

int main(){
    int row , col;
    scanf("%d %d",&row,&col);
    int mat1[row][col], mat2[row][col], sum[row][col];

    // set mat1
    for(int i = 0; i < row; ++i){
        for(int j = 0; j < col; ++j){
            scanf("%d", &mat1[i][j]);
        }
    }

    // set mat2
    for(int i = 0; i < row; ++i){
        for(int j = 0; j < col; ++j){
            scanf("%d", &mat2[i][j]);
        }
    }

    // sum
    for(int i = 0; i < row; ++i){
        for(int j = 0; j < col; ++j){
            sum[i][j] = mat1[i][j] + mat2[i][j];
        }
    }

    // PRINT MAT
    for(int i = 0; i < row; ++i){
        for(int j = 0; j < col; ++j){
            printf("%d ", sum[i][j]);
        }
        printf("\n");
    }

	}

	scanf("%d",&key);

	int found = 0;
	for(int i = 0; i <= (size-1); ++i){
		if(list[i]==key){
			printf("Found at position %d\n",i);
			found=1;
		}
	}
	if(!found){
		printf("%d is not found\n",key);
	}
	
}
